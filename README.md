# POS App

A tablet-first Point of Sale interface built with React Native and Expo.

## Features

- **Landscape Layout**: Optimized for tablet use with a locked landscape orientation.
- **Two-Column Design**:
  - **Left**: Cart/Bill section with dynamic totals and tax calculation.
  - **Right**: Menu section with category filtering and grid layout.
- **Menu Management**:
  - Browse items by category (Appetizers, Main Course, Dessert, Drinks).
  - View item details, stock status, and prices.
- **Order Processing**:
  - Add items to cart with quantity control and special notes.
  - Real-time calculation of subtotal, tax (10%), and final total.
- **Modern UI**: Dark theme implementation using NativeWind (Tailwind CSS).

## Tech Stack

- **Framework**: React Native (Expo)
- **Language**: TypeScript
- **Styling**: NativeWind (Tailwind CSS)
- **State Management**: Zustand
- **Icons**: Lucide React Native

## Setup Instructions

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start the App**:
    ```bash
    npx expo start
    ```

3.  **Run on Device/Simulator**:
    - Press `w` for Web.
    - Press `i` for iOS Simulator.
    - Press `a` for Android Emulator.
